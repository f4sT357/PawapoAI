# 🛠️ セットアップガイド - PawapoAI

このプロジェクトは、ローカルLLM（LM Studio）を使用してPowerPointスライドを自動生成するアプリケーションです。以下の手順に従って環境を構築し、アプリを起動してください。

## 1. 事前準備 (LM Studio)

本アプリは、OpenAI互換のAPIを提供するローカル推論エンジンを必要とします。

1. [LM Studio 公式サイト](https://lmstudio.ai/) からインストーラーをダウンロードし、インストールしてください。
2. LLMモデル（例: `Hermes-3-Llama-3.1-8B`, `Llama-3.2-3B-Instruct` など）をダウンロードし、セットアップしてください。
3. LM Studioの **Local Server** タブ（サーバーアイコン）に移動し、**Server On** ボタンをクリックします。デフォルトのポートは `1234` です。
4. **Base URL** が `http://localhost:1234/v1` であることを確認してください。

---

## 2. Python 環境の構築

Python 3.10 以上がインストールされていることを確認してください。

### ① インストール
```bash
# 仮想環境の作成
python -m venv .venv

# 仮想環境の有効化 (Windows)
.venv\Scripts\activate

# 仮想環境の有効化 (Mac/Linux)
# source .venv/bin/activate

# 必要なライブラリのインストール
pip install -r requirements.txt
```

---

## 3. アプリケーションの起動

### ① アプリの実行
```bash
streamlit run app.py
```

ブラウザが自動的に開き、アプリのメイン画面が表示されます。

### ② 接続の確認
サイドバーの **LLM 設定** セクションで **使用モデル** が表示されていることを確認してください（LM Studioと正しく接続されている場合、ロード済みのモデルがリストに表示されます）。

---

## 📂 各フォルダの役割

- **`data/`**: 解析対象のドキュメント（PDF, TXT, HTML）を保存します。
- **`templates/`**: PowerPointのテンプレート（.pptx）を保存します。
- **`schemas/`**: Pydanticを使ったデータのバリデーション定義です。
- **`services/`**: LLM呼び出し、ファイル解析、PPTX作成などの主要ロジックが含まれます。

---

## ❓ トラブルシューティング

- **モデルが表示されない**: LM Studio のローカルサーバーが **ON** になっているか、ポート番号が `1234` になっているか確認してください。
- **PPTXエラー**: テンプレートファイルが破損しているか、対応していない形式の可能性があります。新規スライドで試してみてください。
- **エンコーディングエラー**: Windows環境で日本語パスを含む場合、文字化けや読み込み失敗が発生することがあります。パスに日本語を含まない場所での実行を推奨します。
